# Официальная серия bake-off — 2026-07-04

Машина: AMD Ryzen 7 8700F (8 физ./16 лог. ядер), NVMe, Windows 11. `--threads 8` у всех.
Состояние: Docker/WSL **остановлены**, фон минимален.
**Отклонения от протокола**: без прав администратора — powercfg High Performance и исключения Defender не применены. Следствие зафиксировано: в раунде 1 первый прогон Go дал 33.8 с при 2.3% CPU (Defender сканировал свежий exe и корпус при первом касании) → раунд 1 признан невалидным по правилу разброса >10%, засчитан раунд 2.

| Участник | Wall, с | Peak RSS, МБ |
|---|---:|---:|
| **C++ core** | 9.99 | **2 350** |
| Rust | 10.72 | 14 963 |
| Go | 9.19 | 16 843 |

Диск-баунд (~750–830 МБ/с), но **память различается в 6–7 раз**: C++ читает скользящим окном mmap 64 МБ, Go/Rust держат файл целиком (окно допуска ограничивает число файлов, не байты). На хосте с 32 ГБ два таких файла параллельно = OOM у Go/Rust. События: 8 541 591 у всех.

## Серия 3: полный корпус 175 ГБ (7100 файлов, холодный кэш), 1 прогон

| Участник | Wall, с | ГБ/с | События |
|---|---:|---:|---:|
| **Rust** | **96.2** | **1.69** | 121 485 348 |
| C++ core | 98.4 | 1.65 | 121 485 348 |
| Go | 180.4 | 0.90 | 121 485 348 |

Rust и C++ в пределах ~2% (диск-баунд), **Go деградировал вдвое** на холодном большом корпусе (GC-давление от цельночитаемых файлов — согласуется с его RSS). Peak RSS в этой серии неинформативен (WorkingSet включает file-backed страницы, ОС раздаёт свободную память под кэш всем).

Расхождение на заметку: нормализаторы насчитали 121 485 348 событий, в ClickHouse после импорта 121 485 342 строки (−6) — разобрать при внедрении e2e-серии.

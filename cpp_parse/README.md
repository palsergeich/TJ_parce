# C++ Parse - Подсчет контекстов из лог-файлов 1С

Оптимизированная версия программы для подсчета уникальных контекстов из технологических журналов 1С.

## Особенности

- **SIMD оптимизации** (AVX2) для максимальной производительности
- **Memory-mapped файлы** для эффективной работы с большими файлами
- **Многопоточная обработка** с разделением на читателей и парсеров
- **Zero-copy парсинг** для минимизации копирования данных
- **Поддержка UTF-8** для корректного отображения русских символов

## Требования

- Компилятор с поддержкой C++17 и AVX2:
  - Windows: Visual Studio 2019+ (MSVC) или MinGW-w64
  - Linux: GCC 8+ или Clang 8+

## 🚀 Быстрый старт (Windows)

### Вариант 1: Интерактивное меню (рекомендуется)

```cmd
cd cpp_parse
build_simple.bat
```

Выберите компилятор из меню:
1. MSVC (Visual Studio) - лучшая производительность
2. MinGW-w64 - если нет Visual Studio
3. CMake - если установлен CMake

### Вариант 2: Прямая сборка с MSVC

```cmd
cd cpp_parse
build_msvc.bat
```

Автоматически найдёт Visual Studio и скомпилирует.

### Вариант 3: Сборка с MinGW-w64

```cmd
cd cpp_parse
build_mingw.bat
```

Требуется установленный MinGW-w64 в PATH.

### Вариант 4: CMake (если установлен)

```cmd
cd cpp_parse
build.bat
```

## 🐧 Сборка на Linux

```bash
cd cpp_parse
mkdir build && cd build

# С GCC
g++ -O3 -march=native -mavx2 -flto -std=c++17 -pthread \
    -o count_contexts ../count_contexts.cpp

# Или с CMake
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . -j$(nproc)
```

## ▶️ Запуск

После успешной сборки:

```cmd
# Windows
cpp_parse\run_cpp_parse.bat "C:\TJ_Logs\TJ_Logs" 16

# Linux
./build/count_contexts "/path/to/logs" 16
```

Где:
- Первый аргумент - путь к папке с логами
- Второй аргумент (опционально) - количество потоков (по умолчанию = количество ядер CPU)

## Архитектура

Программа использует двухэтапную архитектуру:

1. **Потоки-читатели** - читают файлы и разбивают их на события
2. **Потоки-парсеры** - извлекают контексты из событий

События передаются через оптимизированную очередь с batch processing для снижения contention.

## Формат входных данных

Программа обрабатывает технологические журналы 1С в формате:
- События начинаются с маски времени: `ЧЧ:ММ.СССССС-Длительность,`
- Контексты находятся в полях `,Context='...'`
- Поддерживаются многострочные контексты

## Результаты

Результаты сохраняются в файл `result_cpp.txt` в родительской директории указанной папки.

Файл содержит:
- Статистику обработки
- Список всех уникальных контекстов
- Количество вхождений каждого контекста

## Производительность

- Обработка больших файлов (50+ МБ) через memory-mapped I/O
- SIMD оптимизации для поиска и хеширования
- Параллельная обработка нескольких файлов одновременно
- Оптимизированные структуры данных для минимизации аллокаций

## Установка компиляторов

### Visual Studio
1. Скачайте: https://visualstudio.microsoft.com/downloads/
2. Установите "Desktop development with C++"

### MinGW-w64 (MSYS2)
1. Скачайте: https://www.msys2.org/
2. Установите в `C:\msys64`
3. Откройте "MSYS2 MinGW 64-bit"
4. Выполните: `pacman -S mingw-w64-x86_64-gcc`
5. Добавьте в PATH: `C:\msys64\mingw64\bin`
